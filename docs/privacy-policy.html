Política de privacidad

Festival Sulayr (en adelante, la “Aplicación”) es una aplicación informativa que permite consultar programación y contenidos relacionados con el festival.

Datos que recopilamos

La Aplicación no requiere crear una cuenta ni iniciar sesión.

No solicitamos datos personales identificativos (por ejemplo, nombre, dirección, número de teléfono o correo electrónico) dentro de la Aplicación.

Servicios de terceros

La Aplicación se conecta a servicios externos para obtener contenido (por ejemplo, programación, imágenes u otros recursos). Estos servicios pueden procesar información técnica necesaria para la comunicación (por ejemplo, dirección IP, tipo de dispositivo o datos de red) de acuerdo con sus propias políticas.

Permisos

La Aplicación puede solicitar permisos del sistema únicamente para funciones propias del dispositivo (por ejemplo, abrir enlaces externos o guardar contenido multimedia si el usuario lo solicita). No se utilizan permisos para recopilar datos personales con fines publicitarios.

Publicidad

La Aplicación no muestra anuncios.

Compartición de datos

No vendemos ni compartimos datos personales con terceros con fines comerciales.

Seguridad

Las comunicaciones con servicios externos se realizan mediante conexiones cifradas cuando están disponibles (por ejemplo, HTTPS).
